ACTIVIDAD FORMATIVA SEMANA 03

Según el caso de estudio de la semana 3 una empresa de distribución quiere
calcular cuanto se cobrara por el despacho a domicilio dependiendo de la 
distancia y cantidad de compra por el cliente.

Los requerimientos Funcionales son:

Ingreso del cliente por su nombre apellido y edad
Ingreso del vehículo de despacho (marca, modelo, tipo de combustible. etc)
Ingreso de los datos de despacho (total antes del calculo y distancia a despachar)

Salida de datos de usuario y vehículo de despacho
Salida del total final calculado usando las operaciones según lo expresado en la actividad

Los requerimientos No Funcionales son:

Interfaces para el ingreso de datos.
Interfaces para mostrar los datos.
Diseño de aplicación para ser cambiada según lo que pida el cliente
programa básico usando JVM
